## Frontend Reactjs 
## Getting Started

First, run the development server:

copy .env.sample to env
```bash
cp .env.sample .env
```

```bash
npm install
# or 
yarn install

npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
